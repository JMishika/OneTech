# **App Name**: Univers Tech Showcase

## Core Features:

- Homepage Navigation: Fixed navbar with smooth scrolling to different sections: Hero, Services, About, Contact. CTA button for 'Free Quote'.
- Animated Hero Section: Visually engaging hero section with a subtle CSS gradient animation and particle effects. Prominent title and subtitle, confidence-boosting stats, and clear CTAs.
- Service Grid: A responsive grid showcasing all 19 services offered, each with an appropriate tech icon, descriptive title, reassuring description, and 'Expert Certified' badge.
- About Us Section: Showcase company's background and expertise with a team photo, a brief description highlighting 10+ years of experience, CSS progress bars illustrating skill proficiency, and client testimonials.
- Contact Form & Map: Professional contact form, embedded Google Maps (Kinshasa location), business hours, and a WhatsApp contact button to facilitate easy communication.

## Style Guidelines:

- Primary color: Institutional navy blue (#1E3A8A) to inspire confidence.
- Accent color: Tech blue (#3B82F6) to add a modern tech feel.
- Background color: Desaturated navy blue (#E8EBF2).
- Body and headline font: 'Inter', a sans-serif, providing a serious and technical feel.
- Code font: 'JetBrains Mono' for technical sections (monospace font).
- Utilize precise tech icons from Font Awesome Pro-style (or the free version if Pro is unavailable) to represent services and expertise.
- Micro-animations such as hover scale (1.05) and fade-in on scroll to add subtle interactivity and engagement.
